## ✨ Fitur Utama

### 🔐 Login
Fitur yang digunakan untuk proses autentikasi pengguna agar administrator dapat mengakses sistem sesuai hak akses yang dimiliki.

### 📊 Dashboard
Halaman utama administrator yang ditampilkan setelah berhasil login. Dashboard menjadi pusat navigasi untuk mengelola seluruh data pada website.

### 🌐 Blog Pribadi
Halaman publik yang menampilkan daftar artikel bertema teknologi. Pengunjung dapat melihat informasi artikel dan membaca isi artikel secara lengkap.

### 📂 Kategori
Fitur untuk mengelola kategori artikel. Administrator dapat menambahkan, mengubah, menghapus, dan melihat daftar kategori yang digunakan pada artikel.

### 📝 Artikel
Fitur untuk mengelola artikel blog. Administrator dapat menambahkan artikel baru, mengubah data artikel, menghapus artikel, serta mengunggah thumbnail artikel sebelum dipublikasikan.

### 👤 Profil
Fitur yang digunakan untuk mengelola informasi akun administrator, seperti mengubah nama, email, dan kata sandi.

### 🚪 Logout
Fitur untuk keluar dari sistem sehingga akun administrator tetap aman setelah selesai menggunakan aplikasi.
